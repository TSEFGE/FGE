Nota: Para poder subir el proyecto de esta manera tienes que estar dado de alta como contribuyente
si no te mandara un error 403 permiso denegado, y por ultimo recordad antes de empezar hacer
modificaciones hacer un pull para descargar las últimas actualizaciones del proyecto.

1. Tener instalado Git.
2. Desde la consola ubica tu carpeta donde clonaras el proyecto.
3. teclea: git clone https://github.com/TSEFGE/FGE.git
4. Entra a la carpeta del proyecto cd FGE.
5. Tecleamos git branch -a para ver las ramas locales y remotas.
6. Creamos una nueva rama: git checkout -b develop origin/develop
7. No posicionamos en nuestra rama develop: git checkout develop
8. Realizamos los cambios al archivo.
9. Una vez terminados los cambios tecleamos git status, para verificar el archivo modificado.
10. Tecleamos git add nombre de archivo, para preparar el archivo.
11. Tecleamos git commit -m "Mensaje sobre nuestra modificación".
12. git push origin develop, Para subir las actualizaciones al repositorio.

Nota2: Otra manera de subir el proyecto es con un FORK, que agrega el proyecto a tu cuenta de GitHub,
y una vez realizado los cambios subir y hacer un pull request con el proyecto original.
